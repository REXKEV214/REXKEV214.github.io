PaddleOCR 快捷截图流程

发表于2023-08-12|更新于2025-10-30|自动化脚本
|浏览量:

在 Linux 桌面上频繁做 OCR 抄录太费劲，于是写了一个「截图 → 识别 → 自动打开结果 → 复制到剪贴板」的一键脚本组合。核心拆成三个部分：Python 识别脚本、Shell 自动化脚本、系统快捷键。

1. Python 识别脚本

ocr.py 负责调用 PaddleOCR，并把识别内容保存为文本与标注图。

2. Shell 自动化脚本

ocr.sh 完成截图、放大、OCR、打开结果到复制剪贴板的一条龙：

依赖列表：

3. 绑定全局快捷键

  1. mkdir ~/ocrtemp，将 ocr.py、ocr.sh 放入该目录。
  2. Ubuntu 系统设置 → 键盘快捷键 → 自定义 → 新建。
  3. 名称自取，命令填写 /home/<user>/ocrtemp/ocr.sh，组合键建议设为 Ctrl+Alt+O。

之后按下快捷键即可完成「截图 → OCR → 打开结果 → 自动复制」的全流程。
