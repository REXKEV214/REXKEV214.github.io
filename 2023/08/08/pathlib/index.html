创建路径

几乎所有pathlib 的功能都可以通过其 Path 子类访问，可以使用该类创建文件和目录

处理文件

想要创建文件并写入内容，不必再使用 open 函数，只需创建一个 Path 对象搭配 write_text 或 write_btyes 即可

读取文件，则可以 read_text 或 read_bytes

但请注意， write_text 或 write_bytes 会覆盖文件的现有内容
